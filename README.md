# SwapSwap

An open-source infra which let anyone to swap their USDC/DAI/ETH to the *token*  or vice versa by just sending it to `[token].swapswap.eth`.

This is currently working as an experiment and available only on Base Mainnet. 

## Which tokens can be swapped?

Every token which will meet certain criterea (TBD) will be available for swapping by deploying the contract for the *token* and assigning it the ENS subdomain.
